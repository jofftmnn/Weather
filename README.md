# Weather App 
A simple weather application built using HTML, CSS, and JavaScript.  
It shows the current weather and a 5-day forecast using the OpenWeatherMap API.

## Features
- Search weather by city
- Get weather using current location (GPS)
- Displays temperature, condition, and humidity
- 5-day weather forecast
- Mobile-style UI design

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API
